Create a comprehensive luxury yacht booking mobile application called Nauttec with the following complete specifications:

🎨 Design System & Visual Theme
Color Palette: Ocean blues with elegant whites (#2563eb primary, HSL-based semantic tokens)

Icons: Lucide React icons throughout
Mobile-First: Responsive design optimized for mobile devices

UserProfileScreen (/profile) - User profile and settings access

MyBookingsScreen (/my-bookings) - List of user's bookings
ReservationDetailScreen (/reservation-detail/:id) - Individual booking details
