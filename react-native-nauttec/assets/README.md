# Assets Directory

This directory contains app assets:

- **icon.png** - App icon (1024x1024)
- **adaptive-icon.png** - Android adaptive icon (1024x1024)
- **splash.png** - Splash screen image (1284x2778)
- **favicon.png** - Web favicon (32x32)
- **logo.png** - App logo (used in login screen)

## Required Assets

To complete the app setup, you need to add:

1. **App Icon** (icon.png) - 1024x1024 PNG
2. **Adaptive Icon** (adaptive-icon.png) - 1024x1024 PNG for Android
3. **Splash Screen** (splash.png) - High resolution image
4. **Favicon** (favicon.png) - 32x32 PNG for web
5. **Logo** (logo.png) - Logo for login screen

You can generate these using the Expo asset generation tools or create them manually.
